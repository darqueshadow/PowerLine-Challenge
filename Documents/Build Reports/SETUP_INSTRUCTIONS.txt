HOW IT WORKS:
-------------
1. User clicks the cassette tray zone on the main menu
2. CRT flicker transition plays (500ms)
3. Submenu appears with:
   - Horizontal cassette row (showing the blank cassette)
   - Screenshot panel (showing placeholder)
   - Info panel (showing "Blank Cartridge" details)
   - Cassette deck at bottom with Start/Back buttons

4. Navigation:
   - Click cassettes or use Left/Right arrow keys (or A/D keys)
   - Click "Start" button (currently disabled for blank cartridge)
   - Click "Back" button to return to main menu


ADDING NEW CARTRIDGES:
----------------------
To add a new game cartridge (e.g., "Asteroid Command"):

1. In submenu.js, locate the loadCartridges() function
2. Add a new object to the array:

   {
     id: "asteroid-command",
     displayName: "Asteroid Command",
     status: "available",                                    // or "coming-soon"
     screenshot: "assets/screenshots/asteroid-command.png",
     synopsis: "Space-themed PLC training mission with falling command asteroids.",
     cassetteImage: "assets/cassettes/asteroid-cassette.png" // custom cassette art
   }

3. Create the corresponding assets:
   - Custom cassette PNG (or reuse blank cassette temporarily)
   - Screenshot PNG
