Load Screen Overlay (in index.html):
-------------------------------------
<div id="cassette-load-screen" aria-hidden="true">
    <div class="cassette-load-content">
        <p id="cassette-load-line-1">READY.</p>
        <p id="cassette-load-line-2">LOAD "BLANK CASSETTE",8,1</p>
        <p id="cassette-load-line-3">RUN</p>
    </div>
</div>

CSS Styling:
------------
- Full viewport overlay (100vw x 100vh)
- Black background (#000)
- Green monospace text (#39ff14) with glow
- Hidden by default (opacity: 0)
- Shown with .active class (opacity: 1)
- z-index: 10000 (above everything)

JavaScript Logic (submenu.js):
-------------------------------
1. Get selected cassette object
2. Update load line text with cassette.displayName.toUpperCase()
3. Add .active class to show overlay
4. setTimeout(navigate, 5000) - wait 5 seconds
5. Navigate to cassette's game URL

Exit Screen (in blank cassette):
---------------------------------
Same structure and styling as load screen
Shows in ejectCassette() function
Delays return to console by 5 seconds

LOAD SCREEN TEXT FORMAT
═══════════════════════════════════════════════════════════

For Blank Cassette:
  READY.
  LOAD "BLANK CASSETTE",8,1
  RUN

For future cassettes (e.g., Asteroid Command):
  READY.
  LOAD "ASTEROID COMMAND",8,1
  RUN

Format: LOAD "<UPPERCASE NAME>",8,1
The cassette's displayName is automatically uppercased
