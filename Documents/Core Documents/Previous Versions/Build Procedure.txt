Context from my design document:
- The project is called "PowerLine Command Training System".

  - Screens: Start, Options, Game, Pause, Game Over.
  - HUD: score, accuracy, shields, streak, bonus notifications, command input panel.
- Audio:
  - Background track “Asteroid Command”, MP3, loops continuously; starts at game start, stops on menu, restarts on replay.
