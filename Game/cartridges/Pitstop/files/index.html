How to Play

Post your unit base-to-base around the Niagara Region. Each leg is a three-beat sequence — AP → ENP → BSE — typed as real PowerLine commands. Fast, correct typing is your throttle; a miss bleeds the speed.

PowerLine Command Reference

SWAP — pit tire/fuel swap command — preview only (net-new; the pit consequence layer is authorized separately).

Shift Change

At real shift-change times a radio call warns you a change is coming. After ENP, type LA to stay mobile — a board of units opens. Clear each one through its chain (type <CMD> <UNIT>) before the roster clock catches it, then BSE to finish the leg. The car never stops.

24-hour units (pre-dawn / evening) run the full chain RCAV → OD → SS → BSEH → CAV SS; 12-hour starts (mid-day) come up as SP and begin at SS. NOTE may be added any time after SS. Set Shift Change in Race Options (or NONE to skip). Earlier = harder.
